class everest3.k2.Target(*args, **kwargs)¶

Bases: everest3.containers.Target

A class that stores all the information, data, attributes, etc. for a K2 target de-trended with everest3.

Parameters:clobber_raw (bool) – Overwrite existing raw light curve? Default False.
ID¶

A str or int identifier for this target.

aperture¶

A 2D numpy array of integers of shape (ncols, nrows) with 1’s corresponding to pixels included in the aperture and 0’s to pixels outside the aperture.

cadence¶

A float corresponding to the observation cadence in seconds.

detrend(**kwargs)¶

De-trend the light curve via everest3.pld.detrend().

dvs_layout¶

The DVS layout guide.

dvsfile¶

The full path to the DVS file for this target.

flux¶

The de-trended flux array.

get_aperture()¶

Computes the optimal aperture for this target.

get_raw_data()¶

Downloads the raw data for this target.

logfile¶

The full path to the log file for this target.

mag¶

A float corresponding to the magnitude of this target in the mission band.

mission¶

The mission module (this module, k2).

model¶

The linear de-trending model.

path¶

The full path to the directory where data is stored for this target.

plot_dvs()¶

Plots the raw and de-trended data in the data validation summary.

raw¶

A TimeSeries object containing the raw light curve for this target.

season¶

The K2 campaign number for this target.

The time array,
